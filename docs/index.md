# Image to PDF Maker

Welcome to the official documentation and home for **Image to PDF Maker**.

## Overview
Image to PDF Maker is a fully open-source, privacy-first application that lets you seamlessly convert multiple images into a single PDF document. It works entirely offline and requires zero internet permissions.

## Features
- **Privacy First:** No internet permission, no tracking, completely offline.
- **100% Ad-Free:** A clean, uninterrupted user experience.
- **Drag & Drop:** Reorder your photos effortlessly.
- **Quality Control:** Choose the perfect output resolution.
- **Multi-language:** Supported in over 20 languages!
